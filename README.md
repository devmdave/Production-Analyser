## 💡 Overview  
**Production Analyser** is a desktop application built with **PyQt5** that helps industries monitor, analyze, and optimize their production processes in **real time**.  

It provides:  
✅ **Customizable Tags** for data mapping  
✅ **Dynamic Dashboard Parameters** (up to 8)  
✅ **Fault Tracking** (delay + type)  
✅ **Real-time Data Updates** without UI freeze  
✅ **Night/Light Mode** for better UX  
✅ **Seamless PLC Communication** via **pycomm3**

## 🔥 Features Implemented  
- ✔️ **Tag Manager** → Fully customizable with PyQt5  
- ⚡ **Dashboard Parameters** → Add/remove up to 8  
- 📊 **Improved Analytics** → Pandas DataFrames for efficiency  
- 🌙 **UI Enhancements** → Night/Light mode  
- 🛠️ **Fault Tracking** → Logs fault delay + type  
- ⏱️ **Real-Time Updates** → Threading for smooth performance  
- 🔌 **PLC Communication** → pycomm3 for live data
